# 问题一：

对附件一的图案用上面提到的两种方式进行内部路径的填充，分别尝试两种1mm、0.1mm精度（单位间隔）
使用等高线方式填充除了要控制每根横线之间单位间隔相等外，还要保证线每根段端点距轮廓距离为一个单位间隔。所以我想先找出外轮廓内缩一个单位间隔后的内轮廓，可这不就是第二种填充方式要实现的效果吗？所以我就先从第二种方式入手。这样只要写好一个获取单圈内轮廓的函数，就可以拿到第一种方式复用了。
那么要如何获取单圈的内轮廓呢？ 一开始，我们尝试了取两点间连线的中垂线上的点，进行平移来得到内轮廓点集，但是这样做的效果没有符合预期。最后我们选择依次取连续三个点D1、D2、D3，将D1、D2的连线所在直线沿法线方向向轮廓内平移一个间隔单位得到直线L1，以此法处理D2、D3得到直线L2，将L1与L2的交点加入点集作为内轮廓点集。

  • 主要代码：

求法线斜率

function [k] = get_normal_k(dot_a,dot_b)
%get_normal_k() 获取两点连线的法线斜率
%传入两点坐标([x1,y1],[x2,y2])
k = -1/((dot_a(2)-dot_b(2))/(dot_a(1)-dot_b(1)));
end

求两点平移后点集（下移部分）

function [output_dots] = T_down_by_k(dot1,dot2,x)
%Translation_down_by_k() 沿两点连线法线斜率向下平移x个距离
%传入点坐标,单位间隔([d_x,d_y],x)，返回平移后的x坐标集和y坐标集
k = get_normal_k(dot1,dot2);
L_x = ((1/(k*k+1))^0.5)*x;
L_y = abs(k*L_x);
if(k>0)
    output_dots_x = [dot1(1)-L_x,dot2(1)-L_x];
    output_dots_y = [dot1(2)-L_y,dot2(2)-L_y];
else
    output_dots_x = [dot1(1)+L_x,dot2(1)+L_x];
    output_dots_y = [dot1(2)-L_y,dot2(2)-L_y];
end
output_dots = [output_dots_x,output_dots_y];
end
